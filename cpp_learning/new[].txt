`delete[]` 是 C++ 中用于释放动态分配数组内存的关键字。在动态分配数组内存时，使用 `new[]` 运算符，而在释放这些动态分配的数组内存时，就要使用 `delete[]`。

具体地说，`delete[]` 关键字用于释放通过 `new[]` 分配的数组内存，它会调用数组中每个元素的析构函数（如果有的话），然后释放整个数组所占用的内存空间。

例如：

```cpp
int* arr = new int[10]; // 分配长度为 10 的整型数组内存
// 使用 arr
delete[] arr; // 释放数组内存
```

在这个示例中，`delete[] arr` 语句释放了之前动态分配的整型数组内存。
